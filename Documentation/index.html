Welcome to the SimpleFrame Documentation

The internet is easy to use. Now it is also as easy to create.

The SimpleFrame project is a project aimed to make it easier than ever before to be able to code and develop your own website. It uses simple CSS and HTML code chunks making it easier than ever to copy and paste the code into your project. And the greatest feature of SimpleFrame you ask? The fact that it is released under the MIT License What is MIT?
In short, a open source license meaning that I'm not responsible for any use of the software.

This documentation aims to get you familiar with how to develop your first website using SimpleFrame (As easy as the name suggests) and can be for a complete beginner or for a complete developer wanting to release a quick website.

We plan to do/add the following things before this Framework is 'complete':

  • Multiple Headers and Footers to choose from
  • Complete the folder 'Resources' to make it easier than ever before to get code.
  • Add the SimpleFrame navigation system to the documentation in some shape or form.
